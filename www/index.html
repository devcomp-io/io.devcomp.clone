Setup ===== **Launch your OWN instanceof devcomp.io!** Requirements: * NodeJS `0.10+` * **DEDICATED** [Amazon AWS Account](http://aws.amazon.com/) Locate workspaces: mkdir ~/dev.workspaces cd ~/dev.workspaces Credentials: echo ' #!/bin/bash -e # IAM credentials for AWS account export AWS_USER_NAME="" export AWS_ACCESS_KEY="" export AWS_SECRET_KEY="" # pinf.io credentials export PIO_EPOCH_ID="" export PIO_SEED_SALT="" export PIO_SEED_KEY="" export PIO_USER_ID="" export PIO_USER_SECRET="" ' > devcomp.activate.sh Clone: git clone git@github.com:devcomp-io/devcomp.git devcomp cd devcomp Install: ./install.sh Activate: source bin/activate.sh Provision instance and deploy services: pio deploy License ======= Copyright 2014 Christoph Dorn License: MIT
